# **App Name**: WorldStats 24/7

## Core Features:

- Dynamic Statistics Display: Visualize key global statistics like government spending, debt, and corruption via interactive charts on individual data cards, updated to reflect mock 'real-time' changes.
- Infinite Scroll Data Loading: Automatically load and display more statistical data cards as the user scrolls down the page, providing a continuous browsing experience.
- Interactive Chart Presentation: Each data card presents statistics using Shadcn UI chart components, similar to cryptocurrency market displays, showing trends and values.
- Detailed Card Information: Cards feature a descriptive title (e.g., 'External Debt: USA vs. China'), relevant national flags, a clear graph, and the current aggregated sum of the statistic.
- AI Data Insight Tool: Provide a brief AI-generated summary or key insights for each complex statistical trend displayed on a card, helping users quickly grasp the implications.

## Style Guidelines:

- Color scheme is dark and monochrome, conveying seriousness and focus. The primary brand color, a deep, desaturated blue-grey, provides a stable and professional base (#3E4B5B). The background is a very dark, subtle charcoal grey (#141719) for a 'brutal black' aesthetic, ensuring strong contrast. An analogous greenish-blue accent color (#609191) is used for highlighting key interactive elements and chart trends, providing visual depth while maintaining the monochromatic feel.
- Both headline and body text use 'Inter', a clean and objective grotesque sans-serif font. This choice supports the data-driven nature of the app with excellent readability and a modern, no-nonsense aesthetic.
- Key iconography includes flags of nations for direct visual association with data points. Other icons, such as arrows indicating trends (up/down) and simple chart-related symbols, are minimalistic and integrate seamlessly into the dark interface, providing clear, functional cues.
- The layout features an infinite scrolling mechanism where data cards load progressively. Each card presents information hierarchically: a prominent title at the top, a visually engaging chart occupying the central area, followed by relevant country flags and the statistical sum. All elements within cards and the overall interface will have subtle, 'pleasant radii and roundings' to soften the brutalist dark theme, providing a contemporary and inviting feel.
- Subtle and fluid animations are employed for smooth loading of new data cards during infinite scrolling. Charts incorporate gentle transitions and interactive hover effects to highlight specific data points, enhancing user engagement and data comprehension.
